> **“在神奇森林里，任何人都能成为任何人” —— 甚至是轻松搞定复杂长文本的学术与商业大神。**

本项目是一款基于 **Multi-Agent（多智能体）架构**的复杂长文本智能拆解与逻辑重构系统。产品采用情感化、动漫风的视觉交互设计，通过合理的工程流编排，旨在解决高认知负荷场景下的非结构化文本阅读效率痛点。

---

## 🎯 产品定位与核心痛点

在日常处理海量前沿外文文献、深度行业研报或复杂的招股说明书时，高频面临以下“阅读效率瓶颈”：

1. **语言与专业壁垒高**：跨语种、高密度的专业术语及公式导致阅读阻力极大。
2. **信息噪音严重**：关键线索、解法和核心逻辑骨架常常隐藏在几万字的冗长段落中。
3. **结构化脉络难梳理**：读完长文后难以快速建立全局视角的逻辑脑图。

**神奇森林智能卷宗处**通过引入 AI Agent 协同工作流，将单篇数十页的复杂文档拆解提炼时间从 **45分钟压缩至45秒**，实现认知与阅读效率的数量级跃升。

---

## 🧠 多 Agent 协同架构 (Workflow Orchestration)

本项目摒弃了传统的“单次 Prompt 粗暴问答”模式，在底层架构上将复杂任务拆解，抽象出四个各司其职的智能体，模拟团队流水线作业，大幅降低模型幻觉：

* **🕵️ 解析警员 (Parser)**：负责 PDF 文本流的精准提取与分块清洗，确保底层信息完整、无死角。
* **🦉 逻辑神探 (Logician)**：核心推理模块。深度下钻文本，精准剥离出“一句话痛点”、“核心创新机制/商业模式”与“证据论证链（论点+数据支撑）”。
* **🦊 总结专家 (Summarizer)**：负责对高度结构化的专业内容进行“降维翻译”，转化为更直观的业务表达。
* **🦥 整理警司 (Formatter)**：在前端极速生成规范的 Markdown 笔记，无缝对接个人第二大脑知识库。

## 💡 业务场景延展 (Business Scenarios)

本系统的底层逻辑具备极强的通用性，核心产品能力可直接平移并赋能于**金融、法律、商业分析**等行业场景：

* **智能研报拆解**：自动提取券商深度研报中的核心财务指标、行业竞争格局与风险提示。
* **招股书/财报摘要**：对数百页的 IPO 文件、公司年报进行一键结构化梳理，自动提取关键线索。
* **合规文档审查**：快速比对和提取冗长法律合规条文中的约束条件，大幅降低人工审核成本。

## 🚀 极速启动 (Quick Start)

### 1. 环境准备

确保您的本地环境已安装 Python 3.8+。建议使用虚拟环境：
`python -m venv .venv`
`source .venv/bin/activate`

### 2. 安装依赖

`pip install streamlit PyMuPDF Pillow`

### 3. 本地资产配置与运行

在项目根目录下，放入您喜爱的两张正方形卡通图片，分别重命名为 `rabbit.png` 和 `fox.png`，即可激活交互界面：
`streamlit run app.py`
